Detailed Description

This module provides access to networking sockets API.

TCP and UDP protocols are supported for both "client" and "server" operations. Implementation uses async IO underneath, but all operations are always done on the caller thread, effectively providing a synchronous API.

Socket can be closed manually to release system resource, otherwise it will be closed during the shutdown.

Usage example:

// open socket
var socket = new Socket({
ipAddress: "127.0.0.1",
tcpPort: 21,
protocol: "TCP",
role: "server",
timeoutMillis: 500
});
// read data from seocket, specify 'bytesToRead'
// to read the exact number of bytes
var received = socket.read({
timeoutMillis: 100
});
// write to socket
socket.write({
data: "HELO",
timeoutMillis: 100
});
// close socket
socket.close();

Reads the data from this socket. If bytesToRead parameter specified - then the result, returned from this function without throwing the exception will always contains specified amount of bytes.

If bytesToRead parameter is not specified - arbitrary number of bytes of data may be returned and 0 bytes are returned on timeout.
